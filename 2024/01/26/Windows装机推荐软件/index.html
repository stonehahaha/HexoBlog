Windows装机推荐软件

发表于2024-01-26|更新于2024-01-26
|阅读量:

安全工具

  1. 火绒 官网

    windows自带的安全管理太极端了，很多文件直接杀死，误拦截很多；360的话其实挺适合不太懂电脑的，一些病毒链接或者下载的危险应用360还是可以拦截的。但是我个人比较喜欢安静省心的，所以选择了火绒，安安静静呆在后代，偶尔用一下拦截弹窗功能还是很香的

  2. Geek Uninstaller 网盘

    GeekUninstaller正式版是一款相当不错的单文件绿色卸载软件，它体积轻巧，反映迅速，并且完全免费，支持多国语言，完美支持支持32位与64位系统，并且它只有一个exe文件 (软件大小不到2M)，你不需要安装即可直接卸载软件、强制彻底清除软件残留垃圾文件或是查看某软件的注册表项目等等，让你的系统瞬间变得高速流畅

  3. Bitwarden 官网

    开源轻量级密码管理器，带有免费云同步

系统工具

  1. AIDA64 官网 网盘地址

    AIDA64极致版拥有同类产品中无与伦比的硬件检测引擎。 它提供有关已安装软件的详细信息，并提供诊断功能和超频支持。由于它是实时监控传感器，它可以收集准确的电压、温度和风扇速度读数，同时其诊断功能有助于检测和防止硬件问题。它还提供了几个基准，用于测量各个硬件组件或整个系统的性能。 它兼容所有32位和64位Windows版本，包括Windows 10和Windows Server 2016

  2. 360驱动 官网 网盘地址

    百万级驱动库，驱动安装一键化。有轻巧版本下载，也有网卡版，适配新机无网络情况

  3. 卡饭输入法 官网

    来自卡饭论坛，集成拼音和五笔，纯净无广告

  4. 腾讯桌面整理 官网

    免费好用的桌面整理工具，记得下载下面一行小字的桌面整理独立版，否则就下载了腾讯电脑管家了

  5. Cloudbox 官网 网盘

    集合几百款常用绿化软件，全部无限制

  6. Honeyview 官网

    一款快速的图片查看器

  7. uTools 官网

    类似everything，快捷键「Alt + 空格」，呼出 uTools 搜索框，可以轻松调用多种快捷小工具。我经常拿来睡眠关机调起程序

  8. Shadowsocks 官网

    SOCKS5代理工具

  9. Clash for Windows 网盘

  10. ClashX for Mac 网盘

  11. Clash for Android 网盘

  12. V2rayN 网盘

  13. f.lux 官网

    自动调节屏幕色温以匹配环境亮度

聊天工具

  1. QQ 官网
  2. 微信 官网
  3. KOOK（原开黑啦） 官网

音乐

  1. 网易云 官网

  2. Listen1 官网

    一站式免费音乐集合，可以搜索和播放来自网易云音乐，QQ音乐，酷狗音乐，酷我音乐，Bilibili，咪咕音乐网站的歌曲，让你的曲库更全面

压缩工具

  1. Bandizip 官网 网盘

    支持Win与Mac，支持压缩和解压多种主流格式

  2. 7zip 官网

    免费好用的压缩工具

  3. 360压缩 官网

    360为数不多良心应用

浏览器

  1. Edge 官网

    装系统都会自带，除非用的是精简过的系统镜像，现在用了Chromium内核后挺不错的，集成了Copilot，不过咱们用不到就是，不过可以通过修改地区+🪜的方式使用

  2. Chrome 官网

    集成了谷歌的应用，但是插件以及内核，Edge已经赶上来了，况且占用还很大

  3. 百分浏览器 官网

    Chrome的替代品，不过更新非常慢，目前一年一更。主打一个本土化，更多小功能，更小的占用

录像截图

  1. OBS 官网

    用于视频录制和直播的免费开源软件。在 Windows、Mac 或 Linux 上快速轻松地下载并开始流式传输。广泛应用于录屏以及直播

  2. 班迪录屏 官网 网盘

    录制的视频文件体积小，录制的同时支持添加logo到视频中当做水印，但是免费版本在上方会有产品官网链接水印，以及录屏十分钟的限制

  3. PixPin截图 官网

    功能强大使用简单的截图/贴图工具，帮助你提高效率。感觉会成为老牌Snipaste的替代品，功能都做的很像，能够长截图甚至能够截动图

  4. Snipaste 官网

    老牌截图软件了，微软应用商店也能下载到，钉图、标注等等都能做到

媒体播放

  1. PotPlayer 中文网 网盘

    PotPlayer软件自带专业的编解码器，方便快捷，一同安装后可观看任何格式视频文件，功能非常强大。PotPlayer播放器启动速度快，播放稳定，还支持给视频加字幕，设置个性皮肤等

  2. KMPlayer 中文网 网盘

    KMPlayer是4K、8K、60FPS等文件播放的强有力播放器。使用更强大的视频渲染器，可以屏幕旋转和播放高品质视频。HEVC（h.265），h.264、VP9等最新品质视频也能使用加速硬件功能，CPU占有率非常低

办公软件

  1. Office 官网

    有钱的可以支持正版。没钱的我选择下方工具，当然了，下载的也是正版，激活方式在下方的激活Windows系统中可以找到

  2. Office Tool Plus 官网

    一个强大且实用的 Office 部署工具。系统要求：Windows 10 或更高版本，不支持其他操作系统（例如 Linux 或 macOS）。在这里面，可以很轻松得选择自己要装的Office工具，里面也自带了激活方式，但是我觉得太繁琐了，不如用下面的一键激活Windows和Office

  3. HEU KMS Activator 网盘 密码：9xum Github

    神器不多说

  4. Draw.io 官网

    跨平台免费的流程图绘制软件

  5. PDF24 官网

    号称为所有PDF问题提供解决方案。官网也提供了很多在线PDF小工具

  6. Koodo Reader 官网 Github

    一款开源高颜值的阅读器，支持电子书以及漫画的常见格式

  7. Pot Github

    一个跨平台的划词翻译软件，支持很多接口，可以划词翻译，OCR截图翻译

写作软件

  1. Flowus 官网

    以云端笔记为载体，配合在线文档、知识库、文件夹等多形态功能，为个人和团队提供数字信息管理与协同的一站式工作中心

  2. Evernote（印象笔记国际版）官网

    多功能云笔记，免费同步两台设备

  3. Typora 官网 网盘

    个人感觉最清爽简洁的markdown笔记软件，所见即所得

编程应用

  1. Tabby 官网

    Tabby是一款可无限定制的跨平台终端应用程序，适用于本地shell、串行、SSH和Telnet连接。还需有sftp功能，界面还美观

  2. Xshell & Xftp 官网

    Xshell是Netsarang Computer推出的的Secure Shell、Telnet和Rlogin虚拟终端。Xshell可以为每个主机创建单独的会话，用户可以通过标签查看每个会话，方便同时操作和管理与主机的会话

    个人的话，只要填一下邮箱就可以激活了

  3. IntelliJ IDEA 官网

    应用最广的代码编辑器，激活码的话，可以看热心大佬

  4. VSCode 官网

    前端用得比较多，不过重点是微软免费，甚至可以当做markdown文本编辑器

  5. Sublime Text 官网

    现代 UI 设计的文本编辑器，很简洁轻巧

游戏相关

  1. Steam 官网

    最大在线游戏零售平台

  2. Epic 官网

    我永远爱E宝
